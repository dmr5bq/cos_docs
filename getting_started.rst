Getting Started 
****************

Welcome to the Center for Open Science's development documentation. This is the place to start if you want to contribute to the Open Science Framework (also referred to as the OSF.) The pages below will help you install and run the OSF on your machine; if you run into any trouble, check out the "Help and Troubleshooting" pages in the menu.
